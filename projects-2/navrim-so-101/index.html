Navrim SO-101

I built and trained the Navrim SO-101 robot arm system. Its a small AI powered robot arm which you can train to do tasks.

It works with a “leader arm”, which you control, and a “follower arm”, which follows leader. Using this system, you can train the follower to pick up objects and do certain tasks. It can then recreate these tasks using an AI model.
