Wheres My Lair

Where’s My Lair is the current game project I’m working on, and its a silly real time strategy game about being an evil supervillain and trying to take over an unexplored world.

It’s my first 3D game and I used blender to create the models.

The main problem I’m having with this game is focusing more on the art than the gameplay. It started as a random test for procedural generation, and slowly evolved into the strategy game it is today.

I’m still working on it, but I’m hoping to release it by summer 2026.
