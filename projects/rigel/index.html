Rigel

Rigel is a 2D space shooter about fighting off an evil alien race, similar to asteroids.

It was made ground up using the Godot Game engine. It took me about a year to make it and it was finished March 2025. You can play it on itch.io.

It was my first game with Godot, and I learned quite a bit while doing so. I taught myself GD Script, instantiation, shaders, particle systems, and a little bit of blender for the backgrounds.
