Zachcade

Using a kit made from laser cut plywood, a raspberry pi and arcade parts, I put together this miniature arcade cabinet, called Zachade.

Its running the RetroPie, a package containing emulation software for Raspberry Pi computers. I configured it to run arcade games with Mame, NES, SEGA Master, and N64 games. It was made from a kit with laser-cut plywood pieces.

This is the first “big” project I did, combining a bunch of skills. I used mild woodworking skills to glue these pieces together, graphic design skills for the intro video and signs, and some soldering for the electronics (which I’m still not very good at).

All in all it was a fun project and I think it adds a lot of life to my room
